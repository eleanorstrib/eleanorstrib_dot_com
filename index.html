About

Hi, I'm Eleanor!

I am a product manager and developer with a passion for using software to help solve difficult human problems.

Using software and data, I believe we can build tools to help us make better decisions and our interaction with technology more intuitive, conversational and accessible.

In my ten years in the tech industry, I've had leadership roles in product management in a tiny startup, a once-tiny startup that went public, and large multinational company, building software rooted in data applications for the ad tech, energy, and HR analytics.

Speaking at tech conferences is something I really enjoy, and have recently been on stage at DjangoCon US, Codeland and PyCon US.

Currently I'm leveraging my tech, analytics and programming background to develop my expertise in AI: I completed the Deep Learning Foundation Nanodegree in 2017 and am now working towards the Natural Language Processing Nanodegree at Udacity.

I'm an alum of the University of Toronto and the Massachusetts Institute of Technology.

Skillset

Here's what I bring to the table.

Product Management

My approach, derived in part from my years in client service and research jobs with a dash of sales, is to start with a user and their challenges and translate their input and supporting data into features.

Programming

Building things has made me a better product manager, with the added bonus that I can work through implementation and architecture questions, build prototypes, even write some production code if the need arises.
Favorite languages and tools are Python, Javascript, Django, Postgres, nltk, ChartJS, Bootstrap & Heroku.

User Experience Design

UX to me encompasses both the flow and design of the software itself and the overall customer experience. I build products and companies that are actually enjoyable to be around.

A Bossy Sort of Voice

NLP project where I showed gender bias in how Hermione was described relative to the male protagonists in the Harry Potter series of book

In this project, my research partner Caroline Winter and I set out to enumerate and analyze color references and mood in 19th century Gothic novels using Python and visualize the results as a textual analysis tool.

  • Talk at Codeland 2018 in New York City, May 5, 2018

  • Tech stack: Python, NLTK, Matplotlib.

Valeez

Side project where I am developer, designer and product manager.

Valeez is a web application that uses weather data via an API and user inputs about destination, departure and gender orientation of their wardrobe to generate a packing list. Users can create an account and save generated lists for past and future trips.

  • Launched January 2016
  • Tech stack: Django, Python, Postgres, Javascript, HTML/CSS, Bootstrap. Weather Underground API. Deployed on Heroku.

Sign Me Up

Technical presentation on implementing user registration packages given at DjangoCon US 2016 in Philadelphia, PA.

Registration steps to use a web app, as well as login and password reset functionality is a common requirement, but where do you begin implementing this in Django? Walking through an actual case study with code examples in a sample app, novice programmers will learn some tips and tricks for finding and implementing the right framework.

  • Presented July 18, 2016

  • Tech: Django, Python, django-registration-redux package, emails with SendGrid.
